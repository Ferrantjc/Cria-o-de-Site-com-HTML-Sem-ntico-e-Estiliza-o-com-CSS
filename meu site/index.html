O que é Web Design e qual sua relação com Front-End?

Web Design é a área responsável pela criação da interface visual e experiência do usuário em sites. Está diretamente ligado ao Front-End, pois envolve a implementação prática dessas interfaces usando HTML, CSS e JavaScript.

Cite três princípios de design que melhoram a experiência do usuário.

  • Consistência visual
  • Legibilidade e tipografia adequada
  • Navegação intuitiva

Qual a diferença entre Wireframe e Protótipo?

Wireframe é um esboço simples da estrutura de uma página, focado em layout e hierarquia. Protótipo é uma versão mais avançada e interativa, simulando a experiência real do usuário.

Liste três ferramentas usadas para criar protótipos.

  • Figma
  • Adobe XD
  • Excalidraw Whiteboarw

Explique a função da tag <header> e dê um exemplo de uso.

A tag <header> define o cabeçalho de uma página ou seção, geralmente contendo título, logotipo e navegação.

<header>
  <h1>Meu Site</h1>
  <nav>...</nav>
</header>

Qual a diferença entre uma tag em bloco e uma tag inline?

Tags em bloco ocupam toda a largura disponível e criam uma nova linha (ex: <div>, <p>). Tags inline ocupam apenas o espaço necessário e não quebram a linha (ex: <span>, <a>).

Como aplicar CSS externo em um documento HTML?

Utilizando a tag <link> dentro do <head>:

<link rel="stylesheet" href="style.css">

Cite três propriedades CSS comuns e sua função.

  • color: define a cor do texto
  • background-color: define a cor de fundo
  • margin: define o espaçamento externo

O que é DOM e como o JavaScript interage com ele?

DOM (Document Object Model) é a representação estruturada de um documento HTML. O JavaScript interage com o DOM manipulando elementos, atributos e eventos dinamicamente.

Escreva um exemplo simples de código JavaScript que exibe uma mensagem no console.

<script>
  console.log("Olá, mundo!");
</script>
